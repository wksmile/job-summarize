##### 什么是同源策略及限制？
> 同源策略限制从一个源加载的文档或脚本如何来与来自另一个源的资源进行交互。这是一个用于隔离潜在恶意文件的关键的安全机制。
 
> 非同源的限制
- Cookie,LocalStorage和IndexDB无法读取
- DOM无法获得
- AJAX请求不能发送

##### 前后端如何通信？
- Ajax，同源下的通信方式
- WebSocket，不受同源策略的限制
- CORS，支持跨域通信也支持同源通信
